This project demonstrates a fully functional Banking System Database built using MySQL, with a strong focus on ACID properties (Atomicity, Consistency, Isolation, Durability).
It includes complete SQL scripts for creating tables, inserting sample data, and executing real banking transactions such as money transfers.
